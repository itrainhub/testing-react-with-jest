# testing-react-with-jest
利用 jest 测试 react 应用示例项目

## 克隆项目

```bash
$ git clone https://github.com/itrainhub/testing-react-with-jest.git
```

## 安装依赖

```bash
$ cd testing-react-with-jest
$ yarn
# 或 npm install
```

## 打包构建

```bash
$ npm run watch
```

## 运行

```bash
$ npm start
```

打开浏览器，访问：`http://localhost:3000` 即可查看运行效果

## 测试

```bash
$ npm test
```

运行测试任务，练习 Jest 使用
